PROYECTO VIDEOJUEGO.

Este programa esta hecho en PHP y sirve para administrar una lista de videojueos.

¿QUE HACE?

Permite registrar videojuegos nuevos con sus datos (ID, titulo, plataforma, genero y estado).
Muestra la lista completa de los juegos guardados.
Permite eliminar un videojuego buscandolo por su ID.
Tiene un menu interactivo facil de usar.
